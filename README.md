# Fountian of Objects
is a game inspired by Hunt the Wumpus, and is an exercise from the C# player's guide

the game creates a 4x4 Grid, and you're spawned the at the (0,0) position. 
Your task is to find the foutain of objects and enable it once again.
But be warned, there will be dangerous pits nearby that can quickly end your journey.

And while there is no Grid drawn in the game, know that going east and south raises the row and column respectively by 1.

# Commands
-to move in a certain direction, use the "move [direction]" command.
-say for example you wanna move south, use "move south".

-to enable or disable the fountain, just type in "enable/disable fountain"

-to exit, use the "exit" command at the entrance. Tho you can only exit once the fountain has been enabled.

- "help" to bring up a help menu anytime you feel stuck!!

  Good Luck!!
